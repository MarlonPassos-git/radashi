### Usage

Given an array of items return the final item that wins the comparison condition. Useful for more complicated min/max.

```ts
import * as _ from 'radashi'

const gods = [
  {
    name: 'Ra',
    power: 100,
  },
  {
    name: 'Zeus',
    power: 98,
  },
  {
    name: 'Loki',
    power: 72,
  },
]

_.boil(gods, (a, b) => (a.power > b.power ? a : b))
// => { name: 'Ra', power: 100 }
```

- Finding the maximum or minimum value in an array using custom comparison logic.

- Implementing algorithms that require pairwise comparisons, like tournament brackets or priority queues.
- Extracting representative values from datasets, such as the "winner" or "best" item based on defined criteria.
- Applying complex comparison strategies to refine selections, such as filtered data sets or dynamic decision-making processes in applications.
